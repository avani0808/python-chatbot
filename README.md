# Python Chatbot

A simple chatbot developed using Python that can interact with users, perform basic mathematical operations, tell jokes, and provide motivational responses.

## Features
- Basic text-based conversation
- Calculator functionality (addition, subtraction, multiplication, division)
- Random joke generation using the random module
- Simple motivational responses
- Continuous interaction loop until user exits

## Technologies Used
- Python
- Built-in random module

## How to Run
1. Download or clone the repository
2. Open terminal or command prompt
3. Navigate to the project folder
4. Run the following command:
   python chatbot.py
